# Bitsquatting the ENS Domain Names
The helper script [`bitsquatting.py`]() generates permutations of an ENS domain that differ by 1-bit from the original domain.

## Example
*Original ENS Name:* `wagmi.eth`

*Permutations:*

- wagmi.eth
- gagmi.eth
